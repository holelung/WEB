비교 연산자

    - 비교 연산자는 
      좌변, 우변을 비교하여
      비교 결과가 맞을 경우 true,
      아닐 경우 false를 반환한다.
      (boolean 자료형 == true/false)


%, 비교 연산자 이용해서 홀짝 판별기 / 배수, 약수 판별기 만들기

복합 대입 연산자

+=, -=, *=, /=, %=

    - 대입 연산자(=) : 오른쪽 갑을 왼쪽 변수에 대입

    - 복합 대입 연산자
      기존 변수에 저장되어 있는 값에
      새롭게 대입되는 값의 산술 연산 결과를 대입

    ex)
    let temp = 10;
    temp += 5; // temp == 15
    (=> temp=temp+5; 와 같음)

논리 연산자

    &&(AND) 연산자 -> A && B

    - A와 B 모두 true인 경우에만
    A && B 연산결과가 true (나머진 false)

    - 글/말/문제에서
    ~와, 그리고, 이면서, ~부터 ~까지, ~사이
    포함되어 있다면 AND 연산자 사용!

    ex) 104는 100 이상의 숫자이면서 짝수인가?
    -> (104 >= 100) && (104 % 2 === 0) -> true
  
    || (OR) 연산자 A || B
    
    - A 와 B 모두 false인 경우에 
      A || B 연산결과가 false  (나머지 true)
    - 또는, ~이거나(거나)
      포함되어 있으면 OR 연산자 사용
    
    ex) 4는 10을 초과하거나 짝수인가?
    (4>10) || (4%2===0)

  
    !(NOT) 연산자
    - 논리 값을 반대로 바꾸는 연산자(논리 부정 연산자)

    !true == false
    !false == true

    ex) 
    !(true && false) == true
