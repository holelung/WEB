요소 속성(Attribute) 관련 메서드

    1. 요소.getAttribute("속성명") : 속성 값 얻어오기

    2. 요소.setAttribute("속성명", "속성값") :
      요소에 새로운 속성 추가(같은 속성이 존재하면 덮어쓰기)

    3. 요소.removeAttribute("속성명") : 속성 제거하기

    4. 요소.hasAttribute("속성명") : 속성 존재여부 확인
      -> true, false 반환

요소 속성에 직접 접근

    - DOM 요소 노출되어 있는 속성은 
      "요소.속성명" 으로 얻어오기/세팅하기 모두 가능!

    * 직접 접근 방법
    [장점]
    1. 짧고 간결
    2. Attribute() 관련 메서드보다 속도적 우위
    3. 표준 HTML에 대해 직접 접근 가능
      -> HTML 요소가 실제로 제공하는 속성을 다룰 수 있다.
    
    [단점]
    1. DOM 요소별 속성 파악이 필요
    2. 일관성 부족

class 관련 속성(classList)

    * class 속성 
    1) 모든 HTML 요소가 가질 수 있는 속성
    2) 중복 가능
    3) 한 요소가 여러 클래스를 가질 수 있다

    * 요소.classList
    1) `요소.classList.add("class1" [, "class2", "class3", ...])`
    - 요소에 class 추가( 여러 클래스 한 번에 추가 가능)
    - 요소에 class가 존재하고 있으면 이어서 추가

    2) 요소.classList.remove("class1" [, ...])

    3) 요소.classList.toggle("class1")

    4) 요소.classList.contains("class")

    5) 요소.classList.replace("기존 class","새 class")
