일반 구조 선택자(위치 기준)

    - 형제 관계에 있는 요소 중 특정 요소만 선택하는 선택자
    
    **** 중요 ****
    위치를 기준으로 구분

    [작성법]

    (A: 선택된 요소)
    A:first-child -> 형제들 중 A가 첫 번째로 위치한 요소가 맞을 경우 선택

    A:last-child -> 형제들 중 A가 마지막에 위치한 요소가 맞을 경우 선택
    
    A:nth-child(숫자 또는 수열) -> 형제들 중 A가 n번째에 위치한 요소가 맞을 경우 선택
    
    A:nth-last-child(숫자 또는 수열) -> 형제들 중 A가 뒤에서 n번째에 위치한 요소가 맞을 경우 선택
